HTML - الأساسيات وبناء المواقع

تعلم أساسيات تطوير الويب من الصفر، حيث ستتعرف على لغة HTML التي تعد اللبنة الأساسية لإنشاء صفحات الويب. ستتعلم كيفية استخدام العلامات (Tags) المختلفة لتنظيم المحتوى، وإنشاء صفحات ويب متجاوبة وسهلة الاستخدام.

في هذه الدورة، ستتعرف على أهمية البنية الدلالية (Semantic Structure) في HTML لتحسين تجربة المستخدم وأداء الموقع في محركات البحث (SEO). كما ستتعلم كيفية تضمين الصور، الروابط، القوائم، الجداول، والنماذج لجعل صفحات الويب أكثر تفاعلية.

  • فهم بناء HTML وهيكله الأساسي
  • إنشاء النماذج والجداول والمحتوى المتعدد الوسائط
  • إضافة الروابط والصور والعناصر التفاعلية إلى صفحات الويب
  • تصميم صفحات ويب متوافقة مع معايير الوصول وتحسين محركات البحث
  • دمج HTML مع CSS لتنسيق الصفحات وتحسين المظهر البصري

CSS - تنسيق الصفحات وإنشاء تصاميم احترافية

أتقن فن تنسيق وتصميم صفحات الويب من الصفر حتى الاحتراف! ستتعلم كيفية استخدام CSS لإنشاء تصاميم أنيقة، متجاوبة، واحترافية تناسب جميع الأجهزة.

سنغطي تقنيات متقدمة مثل Flexbox وGrid لإنشاء تخطيطات مرنة، بالإضافة إلى التحريكات والتأثيرات البصرية لجعل موقعك أكثر جاذبية.

  • فهم المحددات والخصائص في CSS لتنسيق النصوص والعناصر.
  • إنشاء تصاميم متجاوبة تدعم الأجهزة المختلفة باستخدام Flexbox و Grid.
  • إضافة تحريكات وانتقالات لزيادة التفاعل مع المستخدمين.
  • استخدام متغيرات CSS لتحسين كفاءة الكود وسهولة التعديل.
  • تطبيق أحدث معايير التصميم مثل Dark Mode و Neumorphism.

JavaScript - برمجة تفاعلية وتطوير مواقع ديناميكية

اجعل مواقعك أكثر تفاعلية وحيوية باستخدام JavaScript! في هذه الدورة، ستتعلم كيفية برمجة صفحات الويب وإضافة ميزات ديناميكية تعتمد على التفاعل مع المستخدم.

ستتعلم كيفية التلاعب بـ DOM، واستخدام الوظائف المتقدمة في ES6+، والعمل مع واجهات برمجة التطبيقات (APIs) لإنشاء تطبيقات ويب متكاملة.

  • فهم أساسيات JavaScript من المتغيرات، الحلقات، والدوال.
  • التفاعل مع عناصر الصفحة باستخدام DOM Manipulation.
  • التعامل مع الأحداث (Events) وجعل الموقع يستجيب للمستخدمين.
  • استخدام ES6+ (مثل الدوال السهمية، القوالب النصية، والمصفوفات المتقدمة).
  • التعرف على Fetch API وAJAX لجلب البيانات من الخوادم وعرضها بشكل ديناميكي.
  • تطبيق أفضل الممارسات في كتابة كود JavaScript قابل للتطوير والصيانة.
